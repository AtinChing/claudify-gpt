Claude’s interface reads like an editorial workspace: warm paper in light mode, soft charcoal in dark mode, and a clay accent rather than electric blue.

Typography

UI chrome stays in Anthropic Sans. Assistant prose shifts to Anthropic Serif with a slightly roomier line-height, which gives responses a calmer, bookish cadence.

Code

Inline snippets use Anthropic Mono, and fenced blocks sit in quiet mineral cards:

const paper = "#f8f8f6";
const charcoal = "#1f1f1e";
